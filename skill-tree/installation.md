# 💻 Installation

### **Installation:**

* Download the SKILL TREE script file from keymaster.&#x20;

{% content-ref url="../download-purchased-assets.md" %}
[download-purchased-assets.md](../download-purchased-assets.md)
{% endcontent-ref %}

* Move the file to the `resources` folder on your server.
* Add `ensure dh_skillTree` to your `server.cfg` file.
* Add sql.sql to your database
* Restart your server.



{% hint style="danger" %}
DO NOT CHANGE RESOURCE NAME
{% endhint %}
