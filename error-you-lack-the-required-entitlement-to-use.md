# ⁉️ Error: "You lack the required entitlement to use"

## <mark style="color:yellow;">Error: "You lack the required entitlement to use"</mark>

#### Solution:

1. **Restart your server and verify your server license key.**
2. **Ensure you are using the correct account** associated with the purchased resource.
3. If purchased on the wrong account, **transfer the resource** via [Keymaster](https://keymaster.fivem.net/).
